# BLMJKLL

**BLMJKYLL** is a [Jekyll](https://github.com/jekyll/jekyll) theme using [Bulma](https://bulma.io) based on [Jekyll Now](https://github.com/barryclark/jekyll-now)

You can read this [quick start](https://github.com/barryclark/jekyll-now#quick-start) section from Jekyll Now repo or this guide by Barry Clark, [**Build A Blog With Jekyll And GitHub Pages**](http://www.smashingmagazine.com/2014/08/01/build-blog-jekyll-github-pages/) to customize this theme.
